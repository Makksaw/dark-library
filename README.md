# Dark-Library

**Tech Stack: HTML, CSS, JS, JSON, Express.js**

## Installation
1. Clone the repository:
```
git clone https://github.com/Makksaw/dark-library.git
```

2. Navigate to the project directory:
```
cd dark-library
```

3. Install the dependencies:
```
npm install
```

## Run the project
```
npm start
```
